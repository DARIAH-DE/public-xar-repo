Public Application Repository

This server hosts a public repository of application packages which can be installed into an eXist database instance. You can install any package by:

  • Downloading the package and uploading it to eXist, either via eXide or the admin web app.
  • Going to the "Package Repository" page within the admin webapp and use the following URL as public repository: http://demo.exist-db.org/exist/apps/public-repo/public/apps.xml
